# 429 n 叉树的层序遍历

给定一个 N 叉树，返回其节点值的层序遍历。 (即从左到右，逐层遍历)。

例如，给定一个  3 叉树:

```
      1
    / | \
   3  2  4
  / \
 5   6
```

返回其层序遍历:

```
[
  [1],
  [3,2,4],
  [5,6]
]
```

说明:

树的深度不会超过  1000。
树的节点总数不会超过  5000。
